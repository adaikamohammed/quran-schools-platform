## 1. 🕌 قسم الوقف الخيري والتمويل (الصفحة الرئيسية)

### الهدف
إضافة قسم جديد `WaqfSection` في الصفحة الرئيسية (`src/app/page.tsx`) بين `HeroSection` و`FeaturesSection` يصف قصة المنصة ودعوة أهل الخير للمساهمة في استمراريتها.

### النص المقترح
```

بدأت فكرة "فرسان القرآن" بعد بناء نظام متكامل للمدرسة القرآنية للإمام الشافعي
وتجربته ميدانياً لعدة أشهر بنجاح مبهر مع أكثر من 300 طالب و 18 فوجاً. 

إيماناً منا بأهمية التكنولوجيا في خدمة كتاب الله، قررنا تعميم هذه المنصة لتكون
وقفاً مجانياً بالكامل لأي مدرسة قرآنية في العالم.

نسأل الله أن يكتب هذا الجهد في ميزان حسنات أخينا فارس عدايكة (رحمه الله)
الذي سُميت المنصة تيمناً باسمه "فرسان القرآن".
```

### تعديل الأرقام في `HeroSection.tsx`
| الحالي | الجديد |
|--------|--------|

| `5,000+ طالب مسجّل` | `+300 طالب فعلي` |
| `100% مجاني` | يبقى كما هو ✅ |

### مربع التمويل
```

  • الباقة الحالية (محدودة):  مجانية (بيانات محدودة)
  • للوصول لـ 50+ مدرسة:     ~30$ / شهر
  • للوصول لـ 500+ مدرسة:    ~100$-200$ / شهر

من يريد دعم المنصة من الجزائر أو خارجها
```

## 2. ⚡ تحسينات صفحة الحصص اليومية (`sessions/page.tsx`)

### أ. زر "تأكيد حضور الجميع" (Bulk Attendance)
- **الموقع:** فوق قائمة الطلاب (جانب عنوان الجلسة)
- **الوظيفة:** تعيين `attendance = "حاضر"` لجميع الطلاب المحملين دفعة واحدة
- **الفائدة:** توفير 2-3 دقائق يومياً للمعلم في كل حصة

```tsx
// زر إضافي يُعرض فوق قائمة الطلاب
<button onClick={markAllPresent} className="btn-secondary text-xs">
  ✓ تأكيد حضور الجميع
</button>
```

### ب. التقييم الجماعي السريع للسورة الموحدة
- **الشرط:** تفعَّل فقط إذا كانت `sessionType === "حصة أساسية"` وتم اختيار `surahId` موحد
- **الفائدة:** تطبيق تقييم "ممتاز" أو "جيد" على جميع الحاضرين بنقرة

### ج. تنبيه الغياب المتكرر مباشرة في سطر الطالب
- إذا كان الطالب قد غاب الحصتين الماضيتين: تُظهر أيقونة 🔴 صغيرة للمعلم
- البيانات موجودة في `dailyRecords` المحملة محلياً

## 3. 📚 تحسينات صفحة إدارة الطلبة (`students/page.tsx`)

### أ. النقل الجماعي للأفواج (Bulk Transfer)
- **آلية التفعيل:** زر "تحديد متعدد" في أعلى الصفحة
- **عند التفعيل:** تظهر مربعات اختيار (Checkboxes) على كل بطاقة طالب
- **شريط ثابت أسفل الشاشة:** يظهر عدد المحددين + زر "نقل المحددين" + زر "إلغاء التحديد"
- **عند النقل:** نافذة `BulkTransferModal` تختار المعلم الجديد وسبب النقل

```
[ ] طالب 1    [ ] طالب 2    [✓] طالب 3    [✓] طالب 4
                                           
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ 2 طالب محدد   [ نقل المحددين ]   [ إلغاء ]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### ب. أزرار الفلترة السريعة (Quick Filter Tabs)
```
[ الكل ] [ مهددون بالغياب ] [ متفوقون ] [ موقوفون ]
```

### ج. مؤشر الغياب في بطاقة الطالب
- إضافة عداد صغير في بطاقة كل طالب يُظهر عدد الغيابات في الشهر الحالي
- يتلوّن باللون الأحمر إذا تجاوز 3 غيابات
